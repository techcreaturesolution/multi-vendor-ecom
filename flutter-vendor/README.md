# flutter-vendor — Vendor mobile app

Flutter app for vendors to manage orders and products from mobile.

## Setup

```bash
flutter pub get
flutter run --dart-define=API_BASE_URL=http://10.0.2.2:5000
```

## Scope

- Login (vendor role only)
- Dashboard (status, earnings summary)
- Products list
- Orders list + book shipment
- Earnings + payouts

New vendor applications must go through the web portal (`web-vendor/signup`)
because that flow collects bank details and KYC documents that this mobile app
does not yet support.

Platform folders (`android/`, `ios/`) are generated by Flutter — run
`flutter create .` once in this directory if they are missing.
