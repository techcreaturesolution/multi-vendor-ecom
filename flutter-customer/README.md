# flutter-customer — Customer mobile app

Flutter app. Minimal MVP that consumes the backend API.

## Setup

```bash
flutter pub get

# For Android emulator pointing to a backend running on host localhost:5000
flutter run --dart-define=API_BASE_URL=http://10.0.2.2:5000

# For physical device, replace with your machine's LAN IP:
flutter run --dart-define=API_BASE_URL=http://192.168.1.x:5000
```

## Scope

- Catalog grid
- Product detail + add to cart
- Cart
- Checkout with Razorpay (test mode)
- Login / signup
- Orders list

Platform folders (`android/`, `ios/`) are generated by Flutter — run
`flutter create .` in this directory once to populate them if they are missing.
